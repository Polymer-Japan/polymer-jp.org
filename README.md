# はじめる

このプロジェクトをローカルにクローンして、開発の準備を整えます。

```bash
git clone git@github.com:Polymer-Japan/polymer-jp.org.git
cd polymer-jp.org
```

依存パッケージを `yarn` でインストールします。

```bash
yarn
```

ローカルサーバーを起動します。

```bash
npm start
> micro: Accepting connections on port 3000
```

http://localhost:3000 をブラウザで開くことができます。
