## Install & Start

```bash
$ git clone https://github.com/Polymer-Japan/polymer-jp.org
$ cd polymer-jp.org
$ npm install
$ npm start
```

- polymer-cli, bower, firebase-toolsはプロジェクト内のものを使用。

## Develop

```bash
$ npm serve
```

## Deploy

```
$ firebase login
$ firebase use [project-id]
$ npm deploy
```
